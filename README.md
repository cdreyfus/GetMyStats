# Install dependencies

run ```npm install ```

# Configuration

You must add a .env file in which you'll set your user id/password/apps to examin:
```
GOOGLE_USER = "" 
GOOGLE_PWD = ""
DATA_STUDIO_URL = ""
APP_MB =""
APP_MAIN = ""
APP_CONTROL = ""
APP_REGUL = "" 
APP_ADMIN = ""
```

# Run script

run ```npm start```
